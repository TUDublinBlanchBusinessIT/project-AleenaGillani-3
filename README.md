# Movie Collection

A simple Laravel app to manage movies, genres, actors and reviews.

## Features

- User login and registration  
- CRUD (Create, Read, Update, Delete) for      Movies, Genres, Actors, Reviews  
- Soft-delete (“Trash”) and restore records  
- Many-to-many link between Movies and Actors  
- Average rating shown for each Movie  
- SweetAlert2 popups for delete confirmation  
- Bootstrap 4 styling with a dark theme
